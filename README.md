# 🔐 Gerador de Senhas em Python

Pequena ferramenta desenvolvida em Python para gerar senhas aleatórias seguras contendo:

- Letras minúsculas
- Letras maiúsculas
- Números
- Caracteres especiais

O usuário pode escolher:

- Quantidade de senhas
- Tamanho das senhas

## 🚀 Tecnologias

- Python 3
- Bibliotecas padrão:
  - random
  - string
